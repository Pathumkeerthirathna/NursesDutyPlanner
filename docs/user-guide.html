User Guide

Nurse Duty Planner

A simple guide to help any staff member understand and use the app confidently.

This guide covers duty scheduling, shift setup, duty types, settings, and the money planning area.

1. What this app is for

Nurse Duty Planner is a mobile-friendly planning app designed to help care teams organize daily duties and review related planning information in one place.

Scheduling

  • View a dashboard of duties
  • Create and update duty plans
  • Manage shifts and duty types
  • Set weekly planning rules

Money Planning

  • Track accounts and balances
  • Record transactions
  • Plan budgets
  • Manage reminders and recurring entries

2. Main navigation

The bottom menu is the fastest way to move around the app.

MenuPurpose
HomeShows the dashboard summary and date-based overview of duties.
DutiesAdd, edit, filter, and review nurse duty assignments.
ShiftsCreate working shifts such as morning, evening, or night.
TypesDefine duty categories and colors for easy identification.
MoneyManage finances, transactions, budgets, reminders, and reference data.
SettingsSet weekly hours, week start and end, overtime rate, and day-off rate.

3. Quick start for new users

  1. Open Shifts and create the shift windows your team uses.
  2. Open Types and add the duty categories you want to assign.
  3. Go to Settings and confirm weekly hours and rate rules.
  4. Open Duties and add duty records for the required dates.
  5. Return to Home to review the dashboard and workload summary.
  6. If needed, open Money to manage budgets, accounts, and transactions.

4. How to use each section

Home dashboard

  • Shows total duties, shift groups, and duty types for the selected date range.
  • You can review trends for this week, last week, this month, last month, or all records.
  • Use it as a quick overview screen for planning and monitoring.

Duty Plans

  • Select Add Duty to create a new duty entry.
  • Enter the duty date, shift, start time, end time, and duty type.
  • Use filters to narrow results by date, shift, or duty type.
  • You can edit an entry by opening it and update it when needed.
  • You can delete a duty if it is no longer required.
  • The page uses paging, so move between pages when reviewing more records.

Shifts

  • Add named shifts such as Morning, Evening, or Night.
  • Set start and end times clearly.
  • Mark a shift as active or inactive.
  • Choose a color to make the shift easier to recognize in the interface.

Duty Types

  • Create labels for different kinds of duties.
  • Assign colors for quick visual recognition.
  • Mark duty types as active or inactive to control current use.

Duty Settings

  • Set the minimum weekly hours target.
  • Choose the official start and end day of the work week.
  • Define overtime and day-off rates.
  • Save changes after updating any value.

Money section

This area works like a compact finance workspace for personal or operational tracking.

  • Overview: View balance, income, expense, and net cash flow.
  • Transactions: Record income, expenses, and transfers.
  • Accounts: Manage bank or cash accounts and balances.
  • Planning: Work with budgets, reminders, and recurring items.
  • Masters: Maintain users, categories, merchants, and tags.

5. Common tasks

To create a shift

  1. Open Shifts.
  2. Select Add Shift.
  3. Enter the shift name, time range, color, and status.
  4. Save the record.

To add a duty plan

  1. Open Duties.
  2. Select Add Duty.
  3. Fill in date, shift, duty type, and times.
  4. Save the duty.

To review finance activity

  1. Open Money.
  2. Choose the user and date range if prompted.
  3. Review the overview cards for balances and cash flow.
  4. Use the tabs to move to transactions, accounts, or planning tools.

6. Good practice tips

  • Create shifts and duty types first before entering many duties.
  • Use clear names and colors so staff can recognize entries faster.
  • Review the Home screen regularly to spot workload changes.
  • Keep inactive items instead of deleting them when you may need them later.
  • Check settings after any policy or rate change.
Tip: If something does not appear correctly, use the refresh or filter reset option and check that the correct date range is selected.

8. Summary

Nurse Duty Planner is designed to make staff scheduling easier to manage and easier to understand. Once shifts, duty types, and settings are configured, the app becomes a simple daily tool for planning and review.
