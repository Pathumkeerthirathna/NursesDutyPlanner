# Nurse Duty Planner User Guide

This guide explains the app in simple language so any user can understand its main purpose and features.

## What the app does

Nurse Duty Planner helps staff manage:
- duty schedules
- shifts
- duty types
- planning settings
- money tracking and budgeting

## Main screens

### Home
Shows a summary dashboard with total duties, shift groups, duty types, and date range insights.

### Duties
Used to add, edit, filter, and delete duty records.

### Shifts
Used to create working shifts such as morning, evening, and night.

### Types
Used to create and manage duty categories with colors and status.

### Money
Used to manage accounts, transactions, budgets, reminders, and master data.

### Settings
Used to define minimum weekly hours, week start and end, overtime rate, and day-off rate.

## Quick start

1. Create shifts.
2. Create duty types.
3. Review settings.
4. Add duty plans.
5. Check the dashboard.
6. Use the money area if needed.

## Common tasks

### Add a shift
- Open Shifts
- Select Add Shift
- Enter name, time, color, and status
- Save

### Add a duty
- Open Duties
- Select Add Duty
- Fill in date, shift, times, and duty type
- Save

### Review money details
- Open Money
- Review overview cards
- Open tabs for transactions, accounts, planning, and masters

## Tips
- Use colors to make shifts and duty types easier to identify.
- Check date filters when records are missing.
- Save settings after making any policy changes.
